 Student Management System

A simple menu-driven Student Management System developed in C. This project allows users to manage student records using structures and file handling for permanent data storage.

## Features

- Add Student
- Update Student
- Delete Student
- Search Student
- Display All Students
- Save Records in File

## Technologies Used

- C Programming
- Structures
- File Handling
- Functions

## How to Run

1. Compile the program.
2. Run the executable file.
3. Select an option from the menu to manage student records.

Example:

```bash
gcc student_management.c -o student_management
./student_management
